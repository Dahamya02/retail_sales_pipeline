# Retail Sales Pipeline

## Project Overview

Retail Sales Pipeline is a data engineering project designed to simulate a real-world retail sales data workflow. The project focuses on generating synthetic retail transaction data and processing it through a structured data pipeline.

The project is being developed as a learning exercise to strengthen skills in Data Engineering, ETL processes, Lakehouse Architecture, and Machine Learning workflows.

---

## Objectives

* Generate synthetic retail sales data using Python
* Build an ETL (Extract, Transform, Load) pipeline
* Implement Medallion Architecture (Bronze, Silver, Gold layers)
* Practice Data Engineering concepts learned from IBM Data Engineering courses
* Prepare data for future analytics and machine learning applications
* Maintain project version control using Git and GitHub

---

## Dataset

The project uses a synthetic retail sales dataset containing:

* Order ID
* Product
* Customer ID
* Transaction Date
* Quantity
* Unit Price
* City
* Region

The dataset is generated programmatically using Python.
